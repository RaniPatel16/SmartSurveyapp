# 📋 Smart Survey App

A beautiful, fully-functional field data collection and surveying application built with **React Native** and **Expo**. Designed specifically for field engineers, architects, and site managers to capture, manage, and export on-site survey data seamlessly.

---

## ✨ Features

*   **📱 Modern UI/UX:** Premium, responsive design with smooth navigation and beautiful card layouts.
*   **📷 Camera Integration:** Capture high-quality site photos directly within the app using `expo-camera`.
*   **📍 Real-time Location:** Fetch and reverse-geocode precise GPS coordinates and addresses using `expo-location`.
*   **👥 Contacts Access:** Browse and import client phone numbers directly from the device address book using `expo-contacts`.
*   **📋 Advanced Clipboard:** Instantly copy survey IDs, locations, and phone numbers, or use the built-in scratchpad using `expo-clipboard`.
*   **🗂️ Survey Management:** Create, preview, filter, and delete surveys. Data is managed globally using React's **Context API**.
*   **🧭 Navigation:** Complex, seamless navigation handled by **Expo Router** (Tabs & Drawer integration).

---

## 🛠️ Technology Stack

*   **Framework:** [React Native](https://reactnative.dev/)
*   **Build Tool:** [Expo](https://expo.dev/) (SDK 54)
*   **Navigation:** Expo Router (`expo-router`)
*   **Icons:** `@expo/vector-icons` (Ionicons)
*   **State Management:** React Context API

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed, and the [Expo Go](https://expo.dev/client) app installed on your iOS or Android device.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/SmartSurveyApp.git
   cd SmartSurveyApp
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npx expo start
   ```

4. **Run on your device:**
   * Open the **Expo Go** app on your phone.
   * Scan the QR code displayed in your terminal.
